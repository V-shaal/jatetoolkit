Usage:

[REQUIREMENT]
Min. 512MB memory
Java 1.5

[GENERAL]

java [memory_config] -classpath [all_jar_files] uk.ac.shef.wit.jate.debug.[algorithm_testing_class] [path_to_corpus] [optional_reference_corpus_path]

The optional reference corpus path needs to specified when you use an algorithm that requires a reference corpus, such
as Weirdness and GlossEx and TermEx. An example reference corpus - the BNC - is distributed with this tool, at "nlp_resources/bnc_unifrqs.normal".

For default folder structure, i.e., after unzip, do the following example:
Example:

java -Xmx512m -classpath "jatr.jar;libs/wit-commons/wit-commons.jar;libs/trove/trove.jar;libs/opennlp-tools-1.3.0/maxent-2.4.0.jar;libs/opennlp-tools-1.3.0/opennlp-tools-1.3.0.jar;libs/opennlp-tools-1.3.0/jwnl.jar;libs/opennlp-tools-1.3.0/trove.jar;libs/opennlp-tools-1.3.0/jakarta-ant-optional.jar;libs/opennlp-tools-1.3.0/jwnl-1.3.3.jar;libs/opennlp-tools-1.3.0/ant.jar;libs/dragon/dragontool.jar;libs/apache-log4j-1.2.15/log4j-1.2.15.jar;libs/text/text.jar;" uk.ac.shef.wit.jate.debug.TestFrequency
     test/animalcorpus

On MAC or Linux, replace ";" with ":"

     
which sets memory to 512MB (minimum), uses default jar file paths, including "jatr.jar" at your running path, all other jar
files in default "lib" folder, uses simple frequency based algorithm (uk.ac.shef.wit.jate.debug.TestFrequency)
and uses corpus at "test/tiny".

For a collection of implemented algorithms and their main classes, see Package uk.ac.shef.wit.jate.debug. A list of methods you can test with is
appended at the end of this file.

[WINDOWS]
If you are using windows you can use the batch file "atr.batch". It assumes the same relative paths as mentioned above,
i.e., the folder structure after you unzipped the software, and takes three arguments:

1. system memory. 
2. term recognition algorithm
3. corpus path.

For example

atr.bat 512 uk.ac.shef.wit.jate.debug.TestFrequency test/tiny (for any algorithms that do not require a reference corpus)
atr.bat 512 uk.ac.shef.wit.jate.debug.TestWeirdness test/tiny nlp_resources/bnc_unifrqs.normal (for an algorithm that needs a reference corpus)

will set system memory to 512mb, use simple frequency based algorithm (first line) and uses corpus at relative path "test/tiny"
